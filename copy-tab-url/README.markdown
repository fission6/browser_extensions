#Copy Tab URL (Chrome Extension)
Adds a context menu (right click) item in order to "Copy Tab URL"

##Installation
Visit <https://chrome.google.com/webstore/detail/fdmeijohjeejdnjhlpcmnhnbcbjchchg> to install the extension
